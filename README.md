# dixittest
implementation of boardgame dixit in node.js. Hosted on heroku.

# known bugs
- flicker at startup


# planned features
- enlarge images on click

# start developing 
0. install node and npm
1. clone
2. npm install
3. npm run npm run serve-watch
4. npm start

# deploy
1. npm run build
2. start server
  a) heroku local
  b) npm run serve
